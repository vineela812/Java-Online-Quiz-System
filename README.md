A Java Swing based Online Quiz System with real-time timer, progress tracking, JDBC database integration, and detailed result review panel.

Advanced Features & Highlights
>>Implemented a timer-based quiz system using javax.swing.Timer for real-time countdown per question.
>>Added automatic question switching when the timer reaches zero.
>>Integrated scrollable review panel using JScrollPane to display large number of questions smoothly.
>>Designed dynamic question progress bar to track quiz completion visually.
>>Developed detailed result review screen showing:
   Questions
   Correct answers
   Selected answers
>>Used responsive Swing layouts (BorderLayout, GridLayout, BoxLayout) for clean and adaptive UI design.
>>Implemented event-driven architecture using ActionListener for smooth user interaction.
